# Full idiomatic-Python curriculum authored (17 lessons)

The complete course now exists: lessons 01–17 plus an `index.html` table of
contents, covering comprehensions, unpacking, truthiness, slicing, iteration,
strings/f-strings, dicts, function arguments, context managers, the data model &
dunders, decorators, generators, dataclasses, exceptions/EAFP, type hints,
pattern matching, and a Zen capstone. Every lesson links the shared stylesheet,
embeds the Web Speech voice-over, and ends with a 3-question quiz; prev/next nav
is wired into a clickable chain.

Implication: the learner has not yet *worked through* any of these — this records
that the material exists, not that it's learned. Future sessions should start by
asking which lesson they're on, use the quizzes as evidence of understanding, and
write a learning record per lesson only once they demonstrate the idiom (per the
"coverage is not learning" rule). Next-lesson decisions should come from quiz
results and follow-up questions, not from assuming linear completion.
