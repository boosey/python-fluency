# Teaching notes

## Learner profile
- Experienced programmer. Knows general programming; learning the *Python delta*.
- Teach by contrast: "in many languages you'd do X; the Pythonic way is Y, because Z."
- Skip fundamentals. Assume loops, functions, types, scope are understood.

## Preferences
- Lessons must include voice-over narration (browser Web Speech API, embedded).
- Short lessons, one tangible win each. Tufte-style, beautiful, print-friendly.

## Conventions for this workspace
- Shared stylesheet: `assets/course.css` (every lesson links it).
- Voice-over component: `assets/narrate.js` + a hidden `#narration` block per lesson.
- Lessons numbered `lessons/0001-slug.html`, incrementing.
